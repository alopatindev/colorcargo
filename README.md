colorcargo
====

A wrapper for `cargo` that makes stack traces readable

```
pip3 install --user -r requirements.txt
./colorcargo.py [cargo's arguments]
COLORCARGO_VERBOSE=1 ./colorcargo.py [cargo's arguments]
```
