colorcargo
====

A wrapper for `cargo` that colorizes stack traces

```
pip install -r requirements.txt
./colorcargo.py [cargo's arguments]
```

Configuration
---

Try to play with
```python
FILTER_BORING_LINES = False
CARGO_PATH = '/usr/bin/cargo'
```
