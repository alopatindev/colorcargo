colorcargo
====

A wrapper for `cargo` that makes stack traces readable

```
pip install -r requirements.txt
./colorcargo.py [cargo's arguments]
```

Configuration
---

Try to play with these lines
```python
VERBOSE = False
CARGO_PATH = '/usr/bin/cargo'
```
